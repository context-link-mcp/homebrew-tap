# homebrew-tap

Homebrew tap for [context-link](https://github.com/context-link-mcp/context-link) — a local MCP Context Gateway for AI coding agents.

## Installation

```bash
brew install context-link-mcp/tap/context-link
```

## Usage

```bash
# Index a project
context-link index --project-root /path/to/your/project

# Start the MCP server
context-link serve --project-root /path/to/your/project
```

See the [main repository](https://github.com/context-link-mcp/context-link) for full documentation, tool reference, and configuration options.

## Updates

```bash
brew upgrade context-link
```
